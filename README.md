# Pecher-Adrien-Devoir-3-CV-HTML-CSS
Projet de création de CV en ligne référencé et agrémenté de CSS. 
L’objectif est de reproduire une maquette fournie en respectant la structure, la typographie, la palette de couleurs et plusieurs effets d’interaction.

Le développement a été organisé en issues et branches :

    issue-1 : initialisation du projet

    issue-2 : intégration des polices

    issue-3 : layout général (header, colonnes, footer)

    issue-4 : colonne gauche

    issue-5 : colonne droite

    issue-6 : header complet (positionnement + contacts)

    issue-7 : footer complet

    issue-8 : effets de survol

    issue-9 : finitions (vérification HTML/CSS, référencement, ... )

Pour chaque issue j'ai réalisé :

    Création de branche : git checkout -b issue-X-…

    Développement + commits

    git push -u origin issue-X-…

    Pull Request sur GitHub → merge dans main

    Suppression de la branche
